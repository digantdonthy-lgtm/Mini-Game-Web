# Mini-Game-Web
Mini Game Hub is a standalone web application that acts as a central hub for playing different mini-games. Each game is designed to be quick, engaging, and easy to play directly in the browser without any downloads.
This project focuses on simplicity, performance, and user experience while demonstrating core web development concepts.

Features-
🎯 Multiple mini-games in one place
⚡ Fast and lightweight (no frameworks used)
🧠 Interactive and responsive UI
🔗 Easy navigation between games
💾 LocalStorage support (for saving scores/progress)
📱 Works on both desktop and mobile browsers

Games Included-
1] Tictactoe
2] Memory game
3] Number Guessing game
4] Reaction time game

Program Structure-
MiniGameHub/
│ 
├── index.html         # Main hub interface 
├── tictactoe.html     # tictactoe game interface
├── guessing.html      # number guessing game interface
├── memory.html        # memory game interface
├── reaction.html      # reaction time game interface
